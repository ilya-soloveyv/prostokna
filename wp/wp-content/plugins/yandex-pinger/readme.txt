== Description ==

Яндекс.Поиск для сайта является простым, но эффективным решением для организации поиска для вашего сайта. Он предлагает релевантные результаты поиска, уточнения, полностью настраиваемый дизайн, индексирование по запросу, статистика и многое другое.
Для работы поиска по сайту необходимо, чтобы сайт был проиндексирован Яндексом.
Роботы Яндекса самостоятельно индексируют сайт. Однако, помимо основного индекса, при поиске по сайту используется дополнительный индекс, специально построенный для вашего сайта.
Вы можете повысить приоритет индексирования некоторых страниц вашего сайта с помощью дополнений для CMS, которые могут отсылать запросы на индексацию автоматически. Дополнение, установленное в CMS, отслеживает изменения на сайте и генерирует запросы на индексацию при создании или изменении документов.

Основная инфорамция о том, что такое плагин-пингер, описана на странице http://help.yandex.ru/site/?id=1125183.
Как настроить плагин - написано тут http://help.yandex.ru/site/?id=1125170.

Дополнительную информацию Вы можете получить с этой станицы: http://site.yandex.ru/cms-plugins/

Yandex Site Search is a simple yet effective solution for adding a search tool to your site. It offers relevant search results, suggestions, a fully customizable design, indexing, stats and much more.
(Please note that Yandex automated indexing robots must first index your site for you to use this tool).
In addition to the main search index, another custom index will be created specially for your website. You can also use the WordPress ping plugin, which automatically sends indexing requests to Yandex, to increase the indexing priority for certain pages of your site. The WordPress plugin monitors changes to your site and generates indexing requests when webpages are created or edited, ensuring that your search results remain comprehensive and up-to-date.

Basic information on what the plugin-pinger is described on page http://help.yandex.ru/site/?id=1125183.
How to configure the plugin for WordPress is described here http://help.yandex.ru/site/?id=1125170.

== Installation ==

1. To install and configure the plug-in for WordPress, open the administrative console WordPress on your site.
2. To install the plug-in, select the menu item Plugins > Add new.
3. On the page Install plug-ins click on the link Download.
4. Select the zip file of the plugin and click Install.
WordPress will report on the result of the installation. If the installation is successful, the page displays the link Activatewith the help of which you can activate the installed plug-in.
5. After you activate the open page of installed plug-ins with a notification about the successful activation of the plugin. In the menu of the administration console appears item Yandex.PDS

Configuring plug-in
After activating the plugin menu in the management console there is a new item Yandex.PDS that opens the settings page of the plugin.
Enter values of parameters and click the Save.

All settings to adjust the plug-in you can get on a site of service Yandex Site Search (http://site.yandex.ru) in the settings of your search on the page "Indexing".
